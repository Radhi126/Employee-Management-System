# Employee Management System

A web-based Employee Management System developed using Flask, MySQL, HTML, CSS and JavaScript.

## Features

- Admin Login
- Employee Management (CRUD)
- Department Management (CRUD)
- Search Employees
- Search Departments
- Export Employee Data to Excel
- Reports Dashboard
- Responsive User Interface
- Duplicate Email Validation
- Duplicate Department Validation

## Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript
- Pandas
- OpenPyXL

## Installation

Clone the repository

```bash
git clone https://github.com/USERNAME/employee-management-system.git
```

Go inside the project

```bash
cd employee-management-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Import

```
employee_management.sql
```

into MySQL.

Update database credentials inside

```
db.py
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
